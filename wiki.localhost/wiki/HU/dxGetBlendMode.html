This function returns the current blend mode for the dxDraw functions. The blend mode is set using dxSetBlendMode

Szintaxis

string dxGetBlendMode ( )

Visszatérési érték

Returns the current blend mode, which can be one of:

  • blend
  • add
  • modulate_add
  • overwrite
