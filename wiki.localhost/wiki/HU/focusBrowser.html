HU/focusBrowser

From Multi Theft Auto: Wiki
Jump to:navigation, search

Ez a funkció megpróbálja fókuszba állítani a böngészőt vagy minden böngészőről leveszi a fókuszt. A fókuszban lévő böngésző megkapja a billentyűleütéseket.

Szintaxis

bool focusBrowser ( browser webBrowser )

OOP Syntax Help! I don't understand this!

Method: browser:focus(...)

Kötelező argumentumok

  • webBrowser: A kivalasztani kívánt böngésző - ha ez nil, akkor minden böngészőről leveszi a fókuszt.

Visszaadott érték

Returns true if the browser was focused or if nil was passed, false if it failed to focus or the browser does not exist.

Példa

Click to collapse [-]
Client

Ez a példa létrehoz egy böngészőt, és fókuszba állítja 

local browser = createBrowser(860, 680, false)
addEventHandler("onClientBrowserCreated", browser,
   function ()
       focusBrowser(source)
   end
)
