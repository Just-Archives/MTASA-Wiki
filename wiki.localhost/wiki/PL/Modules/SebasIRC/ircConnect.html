Ta funkcja/zdarzenie jest dostarczana przez zewnętrzny moduł SebasIRC. Musisz go zainstalować aby jej używać.


Ta funkcja zestawia połączenie do serwera IRC.

Składnia

bool ircConnect(string ip, int port, string nickName)

Wymagane argumenty

  • ip: Adres IP serwera IRC
  • port: Port serwera IRC
  • nickName: Nazwa użytkownika dla bota

Wartości zwrotne

True jeśli połączono, w innym przypadku false
