FR/Server Scripting Events

From Multi Theft Auto: Wiki

Cette page liste toutes les événements disponibles côté-serveur et utilisables dans le "Deathmatch mod".

Pour demander une nouvelle fonction ou un événement, utilisez ce lien.

Les événements côté-client se trouvent ici

La description indique à quel moment l'événement est appelé, cliquer pour plus d'informations.

Clients

  • onConsole (Quand un joueur tape un message/commande dans la console)


Zone de collision (colshape)

  • onColShapeHit (Quand un élément entre dans une zone de collision)
  • onColShapeLeave (Quand un élément sort d'une zone de collision)


Elements

Un élément est une classe générique pouvant représenter n'importe quelle entité. (Joueur, widget, véhicule, son...)

  • onElementDestroy (Quand un élément est détruit)
  • onElementDataChange (Quand une donnée d'un élément est modifiée)
  • onElementColShapeHit (Quand un joueur/vehicule entre dans une zone de collision)
  • onElementColShapeLeave (Quand un joueur/vehicule sort d'une zone de collision)
  • onElementClicked (Quand un client clique sur un élément)
  • onElementStartSync (Quand un élément est synchronisé par le client)
  • onElementStopSync (Quand un élément n'est plus synchronisé par le client)
  • onElementModelChange (Quand le model d'un élément est changé, ex: skin d'un joueur, model d'un véhicule...)


Marqueurs

  • onMarkerHit (Quand un élément entre sur un marqueur)
  • onMarkerLeave (Quand un élément quitte un marqueur)


Pickups

  • onPickupSpawn (Quand un pickup spawn ou respawn)
  • onPickupHit (Quand un joueur touche un pickup)
  • onPickupUse (Quand un joueur reste sur le pickup sans être dans un véhicule)


Joueurs

  • onPlayerBan (Quand un joueur ajoute un ban)
  • onPlayerChat (Quand un joueur utilise le chat)
  • onPlayerPrivateMessage (Quan dun joueur envoie un message via /msg)
  • onPlayerConnect (Quand un joueur tente de se connecter au serveur)
  • onPlayerChangeNick (Quand un joueur change son pseudo)
  • onPlayerLogin (Quand un joueur s'identifie sur le serveur)
  • onPlayerLogout (Quand un joueur se déconnecte de son compte)
  • onPlayerDamage (Quand un joueur subit des dommages)
  • onPlayerJoin (Quand un joueur rejoint le serveur)
  • onPlayerQuit (Quand un joueur se déconnecte du serveur)
  • onPlayerSpawn (Quand un joueur spawn sur le serveur)
  • onPlayerWasted (Quand un joueur est tué ou se tue)
  • onPlayerTarget (Quand un joueur vise un élément)
  • onPlayerVehicleEnter (Quand un joueur entre dans un véhicule)
  • onPlayerVehicleExit (Quand un joueur sort d'un véhicule)
  • onPlayerWeaponSwitch (Quand un joueur change d'arme)
  • onPlayerMarkerHit (Quand un joueur touche un marqueur)
  • onPlayerMarkerLeave (Quand un joueur quitte un marqueur)
  • onPlayerPickupHit (Quand un joueur touche un pickup)
  • onPlayerPickupUse (Quand un joueur utilise un pickup)
  • onPlayerClick (Quand un joueur clique avec le curseur)
  • onPlayerContact (Quand un joueur touche un élément différent du précédent)
  • onPlayerStealthKill (Quand un joueur tue furtivement un autre joueur)
  • onPlayerMute (Quand un joueur est muté)
  • onPlayerUnmute (Quand un joueur est démuté)
  • onPlayerCommand (Quand un joueur tape une commande)
  • onPlayerModInfo (Quand un joueur a modifié certains fichiers)
  • onPlayerVoiceStart (Quand un joueur commence à utiliser le chat vocal)
  • onPlayerVoiceStop (Quand un joueur arrête d'utiliser le chat vocal)
  • onPlayerScreenShot (Quand un joueur prend un screenshot via takePlayerScreenShot)
  • onPlayerNetworkStatus (Quand la connexion d'un joueur est interrompue)


Piétons

  • onPedWasted (Quand un piéton est tué ou meurt)
  • onPedWeaponSwitch (Quand un piéton change d'arme)


Ressources

  • onResourceStart (Quand une ressource démarre)
  • onResourcePreStart (Quand une ressource va démarrer)
  • onResourceStop (Quand une ressource est arrêtée)


Serveur

  • onBan (Quand une IP ou un sérial est bannis)
  • onDebugMessage (Quand un message de d'erreur ou d'alerte apparait dans la console)
  • onSettingChange (Quand les paramètres d'une ressource sont modifiés)
  • onAccountDataChange (Quand les données d'un compte sont modifié via setAccountData)
  • onUnban (Quand un ban est supprimé du serveur)
  • onChatMessageQuand un message est envoyé dans le chat ou lorsque say, teamsay ou me est utilisé)


Véhicules

  • onTrailerAttach (Quand une remorque est attaché à un camion)
  • onTrailerDetach (Quand une remorque est détaché d'un camion)
  • onVehicleDamage (Quand un véhicule subit des dommages)
  • onVehicleRespawn (Quand un véhicule respawn)
  • onVehicleStartEnter (Quand un joueur veut rentrer dans un véhicule)
  • onVehicleStartExit (Quand un joueur veut sortir d'un véhicule)
  • onVehicleEnter (Quand un joueur entre dans un véhicule)
  • onVehicleExit (Quand un joueur sort d'un véhicule)
  • onVehicleExplode (Quand un véhicule explose)


Armes

  • onWeaponFire (Quand une arme est utilisée)
