DgsSetTranslationTable

From Multi Theft Auto: Wiki
Jump to:navigation, search

This function sets translation table.

Syntax

bool dgsSetTranslationTable ( string name, table translationTable )

Required Arguments

  • name: Translation Table Name
  • translationTable: Translation Table

Returns

Returns true if succeed, false otherwise

Example

DGS = exports.dgs
TranslationTable = {
	login = "Login",
	register = "Register",
}

if not dgsTranslationTableExists("EN") then
	dgsSetTranslationTable("EN",TranslationTable)
end
