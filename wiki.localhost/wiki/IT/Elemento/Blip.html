La class blip rappresenta piccole icone o segnalini che possono essere mostrati sul radar del player.

Il type di questa class è "blip".

Sintassi XML

<blip posX="" posY="" posZ="" icon="" color="" dimension="" ordering=""/>

Attributi richiesti

  • posX: Una float che rappresenta la posizione sull'asse X dell'icona.
  • posY: Una float che rappresenta la posizione sull'asse Y dell'icona.
  • posZ: Una float che rappresenta la posizione sull'asse Z (altezza) dell'icona.

Attributi opzionali

  • icon: L'icona del segnalino. Di default è 0.
  • color: Il colore dell'icona in formato HTML (#RRGGBB). Di default è blu.
  • dimension: La dimensione del segnalino. Di default è 0.
  • ordering: L'ordine di altezza del segnalino. Di default è 0.
