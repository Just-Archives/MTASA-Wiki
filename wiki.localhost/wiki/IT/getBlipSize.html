IT/getBlipSize

Questa funzione ottiene la dimensione di un segnalino

Sintassi

int getBlipSize ( blip theBlip )

Argomenti richiesti

  • theBlip: Il segnalino del quale vuoi ottenere la dimensione.

Valori restituiti

Restituisce un int indicante la dimensione del segnalino. Il valore di default è 2.

Esempio

Questo esempio azzera la dimensione di tutti i segnalini a quella di default. 

-- Recuperare una tabella contenente tutti i segnalini che esistono
blips = getElementsByType ( "blip" )
-- Scorre l'elenco, e memorizza i segnalini della tabella nella variabile blipValue 
for blipKey, blipValue in ipairs(blips) do
	-- Recupera la dimensione dei segnalini nella variable 'blipSize'
	blipSize = getBlipSize ( blipValue )
	-- Se la dimensione del segnalino non è già 2 (la dimensione di default)
	if ( blipSize ~= 2 ) then
		-- Imposta la dimensione del segnalino a quella di default
		setBlipSize ( blipValue, 2 )
	end
end
