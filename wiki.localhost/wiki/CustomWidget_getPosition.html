CustomWidget:getPosition

From Multi Theft Auto: Wiki
Jump to: navigation, search

This function returns Custom Widget position on screen.

Syntax

float, float CustomWidget:getPosition([bool Relative])

Other Arguments

  • Relative - Relative of position (if false, coordinates will be absolute). Default - false.

Returns

Returns X and Y coordinate.


Example

local Window = CustomWindow.create(0, 0, 100, 100, "Example", false)

Window:addEvent("onClientGUIClick", function()
	local x, y = Window:getPosition()
	local rx, ry = Window:getPosition(true)
	outputChatBox("The Absolute coordinates are: "..tostring(x).." - "..tostring(y))
	outputChatBox("The Relative coordinates are: "..tostring(rx).." - "..tostring(ry))
end)
