Principais Mudanças

  • Tradução para os menus do MTA
  • Classes OOP
  • Matrizes e Vetores
  • Melhorou significativamente a sincronização de trens
  • As funções relacionadas a áudio agora são compatíveis com os elementos do tipo jogador

Scripting

Novas Funções

Cliente

  • createEffect
  • setEffectSpeed
  • getEffectSpeed
  • setEffectDensity
  • getEffectDensity
  • getLocalization
  • isChatVisible
  • downloadFile
  • isTrainChainEngine

Servidor

  • isBan
  • setBanAdmin
  • setBanReason
  • setUnbanTime
  • getAccountsBySerial
  • getAccountSerial

Compartilhados (Cliente e Servidor)

  • Adicionado um parâmetro bInstant para setPlayerMoney a fim de definir o montante de dinheiro instantaneamente, sem aparecer a contagem/descontagem no HUD
  • toJSON/fromJSON agora lida melhor com dados binários

Novos Eventos

Cliente

  • onClientFileDownloadComplete

Servidor

  • onWeaponFire

Mudanças, correções de bugs

  • getResourceConfig() funciona para recursos adicionados posteriormente a inicialização do servidor
  • Corrigido o bug do veiculo (ID: 570)
  • attachTrailerToVehicle agora suporta trens

Cliente

Novidades

  • Distinção entre Shift, Ctrl e Alt esquerdo/direito
  • Adicionados os parâmetros SettingHUDMatchAspectRatio, SettingAspectRatio para dxGetStatus
  • Suporte de arquivos de áudio usando o Opus Codec nas funções playSound e playSound3D

Correções de bugs e mudanças

  • Foram corrigidos as ocasiões:
    • Corrigido o efeito do dinheiro "contagem regressiva" quando você muda de servidor.
    • Os pedestres ficam invencíveis a tiros quando estão no banco de passageiro.
    • O evento onClientPlayerDamage não ativar para a lata de spray.
    • Satchels não são removidas com resetMapInfo.
    • getPedMoveState retornar falso quando o jogador estiver se movendo agachado
    • guiScrollPaneGetVerticalScrollPosition retornar valores estranhos
    • setPedCameraRotation não funcionar as vezes.
    • Pedestres continuarem atirando depois de sua munição ter acabado.
    • Títulos de rádio não aparecerem as vezes.
    • As músicas da rádio serem puladas quando se estiver passando entre as estações.
    • A função para pular uma faixa de áudio (F5) não funcionar.
    • Veículos caírem de repente pelo mapa.

Servidor

Novidades

  • setElementDimension deve agora fazer efeito para elementos subordinados
  • Erros de módulo estão mais explicados nas mensagens
  • Novos comandos: unloadmodule e reloadmodule
  • Adicionadas armas customizadas para o lado do servidor.

Correções de bugs e mudanças

  • Agora há um limite de 128 caracteres para a função setAccountData.
  • Jogadores com nomes contendo caracteres especiais são verificados de forma correta ao conectar.
  • Corrigidos os membros de times não serem enviados ao cliente se forem definidos pela função onResourceStart.
