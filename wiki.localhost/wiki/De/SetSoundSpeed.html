De/SetSoundSpeed

From Multi Theft Auto: Wiki
Jump to: navigation, search

Diese Funktion ändert die Geschwindigkeit der Wiedergabe eines Sound Elements.

Syntax

bool setSoundSpeed ( element theSound, float speed )

OOP Syntax Hilfe! Ich verstehe das nicht!

Methode: sound:setSpeed(...)
Variable: .speed
Gegenstück: getSoundSpeed

Required Arguments

  • theSound: das Sound Element, von welchem du die Geschwindigkeit ändern möchtest.
  • speed: ein float Wert (Fließkommazahl), welcher die Wiedergabegeschwindigkeit repräsentiert.

Rückgabe

Gibt ein true zurück, wenn die Geschwindigkeit vom Sound Element erfolgreich geändert wurde, false wenn nicht.

Beispiel

function soundFunc()
sound = playSound ( "/sounds/jizzy.mp3",true) -- Spielt einen Sound ab
setSoundSpeed ( sound, 1.2 ) -- Spielt den Sound 20% schneller ab
end
addCommandHandler("play",soundFunc)
