Класс маркер представляет из себя цветные трехмерные фигуры в мире GTA. Есть несколько типов маркеров: короно-образные, гоночные чекпоинты, кольце-образные и т.д. В скриптах маркеры часто используются, чтобы обозначить какое-либо место и включать выполнение какого-то действия, когда игрок заходит на маркер.

Тип элемента данного класса - "маркер".

Размер маркера НЕ может быть изменен из XML файла и по умолчанию равен 4.0.

XML синтаксис

<marker posX="" posY="" posZ="" type="" .../>

Обязательные атрибуты

  • posX: Координата позиции маркера в игре по оси X.
  • posY: Координата позиции маркера в игре по оси Y.
  • posZ: Координата позиции маркера в игре по оси Z.

Вторичные аттрибуты

  • type: Визуальный тип маркера. Типы маркеров:
    • "checkpoint": Гоночный чекпоинт, очень высокий, но не бесконечный. В одиночной игре использовался для обозначения пути гонки.
    • "ring": Светящееся кольцо, обычно используется для самолётов.
    • "cylinder": Цилиндр, представляет собой подобие чекпоинта, но не такой высокий. В одиночной игре использовался для обозначения точек начала миссий.
    • "arrow": Стрелка, представляет собой перевёрнутый к земле конус. В одиночной игре использовался как вход в здания.
    • "corona": Светящийся цветной шар.
  • color: Цвет маркера в HTML формате" Пример: #RRGGBB ", по умолчанию красный.
