Элемент/Объект

From Multi Theft Auto: Wiki
Jump to:navigation, search

Класс "объект" является статической 3D моделью в мире GTA. Ими являются только созданные на сервере модели. Модели, входящие в GTA по умолчанию, объектами не являются. Примеры объектов включают создание моделей, дорог и местностей.

Тип элемента этого класса "object".

XML-синтаксис

<object model="" posX="" posY="" posZ="" rotX="" rotY="" rotZ=""/>

Обязательные аргументы

  • model: ID обьекта - используется для упрощения поиска объектов используя getElementByID() функцию или object browser в редакторе карт.
  • posX: Переменная типа float координата X обьекта.
  • posY: Переменная типа float координата Y обьекта.
  • posZ: Переменная типа float координата Z обьекта.

Дополнительные аргументы

  • rotX: Переменная типа float угол поворота X в градусах.
  • rotY: Переменная типа float угол поворота Y в градусах.
  • rotZ: Переменная типа float угол поворота Z в градусах.

Связанные функции

  • setObjectScale
