AR/setSoundVolume

From Multi Theft Auto: Wiki
Jump to:navigation, search

This function is used to change the volume level of the specified sound element. Use a player element to control a players voice with this function.

شرح الوظيفة

bool setSoundVolume ( element theSound, float volume )

العناصر المطلوبة

  • theSound: .الصوت المراد رفع \ اخفاض مستوى الصوت
  • volume: مستوى الصوت من "0.0" أو "1.0" ء

أستعادة

Returns "true" اذا تم بنجاح "false" اذا لم يتم بنجاح

مثال

Click to collapse [-]
Client
function wasted (killer, weapon, bodypart) -- تعريف
        local sound = playSound("sounds/wasted.mp3") -- "wasted" تشغيل صوت
        setSoundVolume(sound, 0.5) -- تعين مستوى الصوت الى 0.5
end -- أغلاق

addEventHandler("onClientPlayerWasted", getLocalPlayer(), wasted) -- اضافة حدث
