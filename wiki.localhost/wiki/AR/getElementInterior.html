AR/getElementInterior

هذه الوظيفة تقوم بجلب رقم المبنى الموجود به الالمنت.

Syntax

int getElementInterior ( element theElement )

OOP Syntax Help! I don't understand this!

Method: element:getInterior(...)
Variable: .interior
Counterpart: setElementInterior

شرح الارقمنتات

  • theElement: الالمنت او الشيئ الذي تريد جلب رقم المبنى الموجود به

Returns

Returns an int for the interior if theElement is valid, false otherwise.

Example

Click to collapse [-]
Server

هذا الكود يتحقق ان كان الاعب داخل المبنى 0 عند كتابة امر باف8'. 

function AmIOutside ( thePlayer, command )
    if ( getElementInterior(thePlayer) == 0 ) then --ان كان الاعب داخل المبنى 0
        outputChatBox ( "Yes you are outside " .. getPlayerName(thePlayer), thePlayer ) --اخراج كلام بالشات
    else --ان لم يكن
        outputChatBox ( "No you aren't outside " .. getPlayerName(thePlayer), thePlayer ) --اخراج كلام بالشات
    end
end
addCommandHandler ( "AmIOutside", AmIOutside )--عند كتابة الامر باف8 "AmIOutside"
