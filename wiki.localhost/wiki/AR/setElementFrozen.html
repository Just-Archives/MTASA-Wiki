AR/setElementFrozen

From Multi Theft Auto: Wiki
Jump to: navigation, search

هذه الوظيفة تقوم بتجميد العنصر او فك التجميد عنه

Contents

  • 1 Syntax
    • 1.1 Required Arguments
    • 1.2 Returns
  • 2 Example

Syntax

bool setElementFrozen ( element theElement, bool freezeStatus )

OOP Syntax Help! I don't understand this!

Method: element:setFrozen(...)
Variable: .frozen

Required Arguments

  • theElement: العنصر المراد تجميده
  • freezeStatus: true تجميد العنصر , false فك التجميد عن العنصر


Returns

Returns true اذا تم تجميد العنصر , false اذا تم فك التجميد عن العنصر .

Example

Click to collapse [-]
Serverside example

بالشات يقوم بتجميده /FreezMe عندما الاعب يقوم بكتابة امر 

addCommandHandler("FreezMe", -- اضافة امر

	function( player , cmd ) -- صنع وظيفة مع تعريف الاعب والامر

	if not ( isElementFrozen ( player ) ) then -- نتحقق ان العنصر وهو الاعب  غير مجمد

	setElementFrozen ( player , true ) -- نقوم بتجميد العنصر وهو الاعب

	end  -- انهاء للتحقق
	
end -- انهاء للوظيفة

  	) ; -- قوس اغلاق الامر

-- Example By Abdul KariM

ClientSide example

بالشات يقوم بتجميده /FreezMe عندما الاعب يقوم بكتابة امر 

addCommandHandler("FreezMe", -- اضافة امر

	function(  cmd ) -- صنع وظيفة مع تعريف الامر

	if not ( isElementFrozen ( localPlayer ) ) then -- نتحقق ان العنصر وهو الاعب غير مجمد

	setElementFrozen ( localPlayer , true ) -- نقوم بتجميد العنصر وهو الاعب

	end  -- انهاء للتحقق
	
end -- انهاء للوظيفة

  	) ; -- قوس اغلاق الامر
