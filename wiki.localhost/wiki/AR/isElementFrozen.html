AR/isElementFrozen

From Multi Theft Auto: Wiki
Jump to: navigation, search

. ( Freeze ) هذه الوظيفة تمكّنك من التحقق مما اذا كان العـنصر مجمد

Syntax

bool isElementFrozen ( element theElement )

عناصر مطلوبة

  • theElement: العنصر المراد التحقق مما اذا كان متجمد

Returns

  • Returns true اذا كان صحيح , false اذا كانت العناصر المدخلة خاطئه او غير صحيحة او العنصر غير صحيح..

Example

Click to collapse [-]
Client

هذا المثال , هو امر يتحقق مما اذا كان اللاعب متجمد .. يفك التجميد 

function unFreeze ( thePlayer, commandName )

		-- اذا كان العنصر متجمد --
		if ( isElementFrozen ( thePlayer ) ) then  -- thePlayer = العنصر اللذي كتب الامر , وهو اللاعب 
				-- وظيفة إعطاء التجميد للعنصر --
				setElementFrozen( thePlayer, false )  -- false = غير متجمد , true = متجمد
				-- إخراج نص بالشات يفيد بأنه تم فك التجميد --
				outputChatBox ( "Unfreeze successfully *", 255,255,0 )  
			  -- إذا لم يكن متجمد
		else
			  -- إخراج نص بالشات يفيد بأن اللاعب ليس متجمداً  --
				outputChatBox ( "not a element freezed *", 255,255,0 )
				
		end  	-- if إغلاق 
end  	-- function إغلاق
addCommandHandler ( "check", unFreeze )   -- للقيام بالوظيفة المحددة check إظافة امر
