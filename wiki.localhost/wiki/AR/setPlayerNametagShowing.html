AR/setPlayerNametagShowing

From Multi Theft Auto: Wiki
Jump to: navigation, search

هذه الوظيفة تستخدم لحذف واظهار اسم الاعب الذي يكون عليه


Syntax

bool setPlayerNametagShowing ( player thePlayer, bool showing )

شرح الارقمنتات

  • thePlayer: الاعب الذي تريد اظهار\اخفاء اسمه
  • showing: استخدم true\false حتى تخفي\تظهر الاسم

true = اظهار الاسم , false = اخفاء الاسم .

Example

هذه الامثلة تخفي اسم الاعب 

function onResourceStart ( )
    local players = getElementsByType ( "player" ) 
    for key, player in ipairs ( players ) do       -- لوب لكل الاعبين
        setPlayerNametagShowing ( player, false )  -- اخفاء اسمائهم (false)
    end
end
addEventHandler ( "onResourceStart", resourceRoot, onResourceStart )

--مثال اخر

function onPlayerJoin ( )
	setPlayerNametagShowing ( source, false ) --اخفاء اسمه (false)
end
addEventHandler ( "onPlayerJoin", root, onPlayerJoin ) --حدث عندما الاعب يدخل
