RU/forcePlayerMap

From Multi Theft Auto: Wiki
Jump to: navigation, search

Эта функция используется для принудительного показа карты игроку.

Синтаксис

bool forcePlayerMap ( player thePlayer, bool forceOn )

Синтаксис ООП Помогите! Я не понимаю, что это!

Метод: player:forceMap(...)
Переменная: .mapForced
Парная функция: isPlayerMapForced

Обязательные аргументы

  • thePlayer: Элемент игрока.
  • forceOn: Логическое значение, указывающее, должна ли карта быть показана игроку.

Возвращает

Возвращает true, если карта игрока была принудительно включена, false в противном случае.

Пример

Этот пример включает карту игроку с именем "dave" на 10 секунд, если этого не было сделано ранее. 

-- Получаем игрока с именем "dave"
dave = getPlayerFromName ( "dave" )
-- Убедимся, что нашли его
if ( dave ) then
    if not isPlayerMapForced ( dave ) then                  -- если его карта ещё не включена
        forcePlayerMap ( dave, true )                       -- включаем её
        setTimer ( forcePlayerMap, 10000, 1, dave, false )  -- скрываем через 10 секунд
    end
end
