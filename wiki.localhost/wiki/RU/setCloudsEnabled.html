RU/setCloudsEnabled

From Multi Theft Auto: Wiki
Jump to: navigation, search

Эта функция делает облака видимыми или невидимыми. Это полезно для гоночных треков, которые находятся высоко, в облаках, которые могут стать причиной низкого FPS.

Использование

bool setCloudsEnabled ( bool enabled )

Необходимые параметры

  • enabled: величина, показывающая видимы ли облака, или нет. Используйте true, чтобы сделать облака видимыми, или false, если хотите сделать невидимыми.

Что возвращается

Возвращается true, если облака были успешно изменены, или false, если был введен неправильный параметр.

Пример

Click to collapse [-]
Сервер

Этот пример делает облака невидимыми для всех игроков. 

function disableClouds ()
    setCloudsEnabled ( false )    -- Hide the clouds for all players when the resource starts
end
addEventHandler ( "onResourceStart", getResourceRootElement(getThisResource()), disableClouds )
