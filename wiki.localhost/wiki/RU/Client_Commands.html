Эта страница описывает все встроенные команды, доступные для выполнения на клиенте. Данные команды можно вводить напрямую в клиентскую консоль или в чат, поставив перед ними / (слэш). Некоторые серверные команды также доступны из клиента. Аргументы внутри [...] опциональны.

Команды ядра

help

Отображает данный список команд

exit

Выходит из приложения

quit

Выходит из приложения

ver

Выводит версию MTA в клиентскую консоль

time

Выводит в чат текущее время

showhud

Использование: showhud [<0-1>]
Отображает HUD

binds

Выводит все назначенные на клавиши действия в клиентскую консоль

serial

Выводит ваш серийный номер в клиентскую консоль

connect

Использование: connect <адрес> <порт> [<ник> <пароль>]
Присоединяет к серверу

reconnect

Присоединяет к последнему посещенному серверу

bind

Использование: bind <defaults/клавиша> [<up/down>] <команда> [<аргументы>]
Назначает действие на клавишу
Пример по назначению клавиши X на прятанье HUD: bind x down showhud 0
Пример по сбросу всех назначенных действий (со всех клавиш): bind defaults

unbind

Использование: unbind <all/клавиша> [<up/down> <команда>]
Убирает назначенное(ые) действие(я) с клавиши
Пример по убору назначения клавиши X с прятанья HUD: unbind x down showhud 0
Пример по убору всех назначенных команд с клавиши X: unbind x

copygtacontrols

Копирует все управление с GTA - для правильной работы может потребоваться перезапуск

screenshot

Сохраняет снимок экрана (скриншот)

saveconfig

Сразу же сохраняет конфигурацию MTA


Команды при наличии подключения к серверу

disconnect

Отсоединяет от сервера и возвращает в главное меню

shownametags

Использование: shownametags [<0-1>]
Показывает метки ников у игроков

showchat

Использование: showchat [<0-1>]
Показывает окно чата

shownetstat

Использование: shownetstat [<0-1>]
Показывает статистику использования сети

chatbox

Использование: chatbox <0-255> <0-255> <0-255>
Устанавливает цвет чата

textscale

Использование: textscale <0.8 to 3.0>
Устанавливает коэффициент масштаба всех отобразителей текста (textdisplay)

showcol

Использование: showcol [<0-1>]
Показывает колшейпы (colshape) в каркасах для помощи в написании скриптов
Работает только в режиме разработки.
Больше информации: setDevelopmentMode

showsound

Использование: showsound [<0-1>]
Пишет id звуков мира в окнах вывода отладочной инфорации, помогая в написании скриптов с setWorldSoundEnabled
Работает только в режиме разработки.
Больше информации: setDevelopmentMode


Команды, на которые можно назначить клавиши

cleardebug

Очищает окно отладки

chatscrollup

Использование: chatscrollup 1
Пролистывает окно чата вверх

chatscrolldown

Использование: chatscrolldown -1
Пролистывает окно чата вниз

debugscrollup

Использование: debugscrollup 1
Пролистывает окно отладки вверх

debugscrolldown

Использование: debugscrolldown -1
Пролистывает окно чата вниз

voiceptt

Передает голос другим игрокам

enter_passenger

Сажает в транспортное средство в качестве пассажира

radio_next

Следующая радиостанция

radio_previous

Предыдущая радиостанция

radar

Использование: radar [<0-1>]
Показывает карту

radar_zoom_in

Увеличивает масштаб карты

radar_zoom_out

Уменьшает масштаб карты

radar_move_north

Сдвигает карту к северу (наверх)

radar_move_south

Сдвигает карту к югу (вниз)

radar_move_east

Сдвигает карту к востоку (вправо)

radar_move_west

Сдвигает карту к западу (слево)

radar_attach

Закрепляет карту за меткой игрока

msg_target

Использование: msg_target <текст>
Отправляет сообщение указанному игроку

vehicle_next_weapon

Меняет оружие на следующее при нахождении в транспортном средстве

vehicle_previous_weapon

Меняет оружие на предыдущее при нахождении в транспортном средстве

radio_next

Меняет радиостанцию на следующую

radio_previous

Меняет радиостанцию на предыдущую
