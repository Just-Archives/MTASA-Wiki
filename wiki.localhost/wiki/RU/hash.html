RU/hash

From Multi Theft Auto: Wiki
Jump to: navigation, search

Эта функция возвращает хэш указанной строки в указанном алгоритме.

Синтаксис

string hash ( string algorithm, string dataToHash )

Required Arguments

  • algorithm: Строка, которая должна быть одной из: "md5", "sha1", "sha224", "sha256", "sha384", "sha512"
  • dataToHash: Строка хэш данных.

Возвращает

Возвращает хэш данных или false, если был использован недопустимый аргумент.
