Основные добавления / изменения

  • Добавлены пользовательские автомобильные сирены
  • Исправлена проблема с медленной загрузкой файлов карт
  • Исправлена ошибка тайм-аутов при смене карты
  • Исправлены многие проблемы и причины сбоев и подвисания
  • Максимальное число игроков увеличено до 4096
  • Добавлены BASS-эффекты
  • Добавлен аналоговый контроль управления
  • Добавлена синхронизация снарядов
  • Исправлены несколько проблем с заменой пользовательских моделей
  • Поддержка ОС Windows 8 (32- и 64-разрядной)
  • Добавлена возможность создания оружия для пешеходов через функцию создания оружия
  • Улучшена стабильность и добавлены новые возможности Редактора карт
  • Программы установки официальной и "ночной" сборок теперь содержат цифровую подпись

Изменения / исправления

  • Исправлена ошибка "not being able to enter vehicles"
  • Добавлен порядковый приоритет к addEventHandler
  • Исправлена "безмоторность" NRG-500 Variant
  • Добавлена возможность пропустить выбор разрешения сдвоенного монитора (Dual Monitor)
  • Организовано использование положения камеры, а не игрока, при расчете расстояния в playSound3D
  • Добавлены цветовое кодирование и субпиксельное позиционирование в dxDrawText
  • Добавлены возможности создания и изменения текстур по технологии CubeMap и объемных текстур (подробнее: см. dxCreateTexture)
  • Добавлена система кэширования моделей для снижения продолжительности загрузки
  • Добавлено кэширование одежды СиДжея для предотвращения подвисаний игры
  • Ускорена процедура обработки серверных и клиентских событий
  • Добавлены свойства элемента в engineApplyShaderToWorldTexture
  • Добавлен необязательный логический параметр в getElementMatrix ( element, bool )
  • Добавлен Unrar для более плавной процедуры обновления
  • Исправлены периодические ошибки при изменении пользовательских моделей и пикапов оружия
  • Исправлена ошибка задержки отображения улучшенных пользовательских моделей транспорта
  • Исправлена точность попадания в onClientPlayerWeaponFire и добавлены параметры точки выстрела
  • Исправлено ранее полностью неверное направление в Fixed setPedAimTarget
  • Исправлена проблема неверного положения появления пуль.
  • Исправлена периодически неверная прорисовка пользовательских моделей
  • Исправлено отсутствие сброса дистанции пользовательской LOD-модели после выхода
  • Добавлено ограничение размера файла clientscript.log
  • Добавлена возможность использования любого оружия совместно с ракетным ранцем
  • Организовано меньше сбоев таймеров
  • Исправлены сбои некоторых анимаций
  • Исправлено возвращение false от getPedMoveState при движении пригнувшись.
  • Исправлена ошибка остановки ресурса, заменяющего модель объекта (dff) и текстуры (txd), в результате чего текстура объекта становится белой
  • Добавлены пиксельные шейдеры пешеходов
  • Добавлена возможность наслоения нескольких шейдеров на текстуру мира
  • Исправлена ошибка отсутствия динамической библиотеки Windows 8
  • Исправлена setElementPosition, быстро вращавшая внепотоковое транспортное средство
  • Добавлена проверка на ошибки загрузки файлов GTA
  • Исправлены сбои Редактора карт
  • Исправлен наводный транспорт при использовании setVehicleIdleRespawnDelay
  • Исправлена showhud, частично не работавшая до размещения (spawn) игрока
  • Исправлена проблема соединения при использовании доменного имени, начинающегося с цифры
  • Исправлено сообщение об отсутствии шрифта
  • Исправлена getSoundLength, возвращавшая 0 для звуковых потоков (не радио-потоков)
  • Исправлен сбой при создании снимка экрана, сворачивания и последующего восстановления окна

  • Исправлен сбой сервера при буксировке транспорта на стороне клиента

  • Большое количество прочих исправлений, улучшений и расширений

Ресурсы

  • Обновлен парашют для устранения высокой нагрузки на центральный процессор клиента
  • Добавлены Ресурсы визуализатора
  • Добавлен Редактор сирен GUI
  • Исправлено использование перезарядки, при котором можно было совершить прыжок и следом перезарядить оружие для мгновенной перезарядки
  • Восстановлена перезарядка оружия при использовании ракетного ранца
  • Исправлены закладки Freeroam
  • Устранены некоторые ошибки и предупреждения

Редактор

  • Добавлена возможность изменения прозрачности объектов и транспорта
  • Изменен выбор цвета транспортного средства для использования новой системы RGB-цветов
  • Исправлено неотображение вкладки серверных настроек в Параметрах при запуске Редактора карт из главного меню
