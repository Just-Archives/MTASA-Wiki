RU/Getting MTA:SA to work with wine

Вам понадобится последняя версия Wine.

Установите MTA:SA, абсолютно не важно, какой у Вас путь к MTA и к GTA:SA.

Вам нужен Wine версии 0.9.49 или более поздней для запуска MTA:SA.

Как только установите его, скачайте Загрузчик для *nix, и извлеките его в папку MTA San Andreas.

Скачайте шрифты verdana, tahoma и tahomabd нужные для CEGUI, и скопируйте их в $HOME/.wine/drive_c/windows/fonts/

Откройте 
winecfg
, перейдите на вкладку "Графика", и снимите флажок "Разрешить Пиксельные Шейдеры".

Удалите из вашей папки MTA San Andreas файл MTA/data/menu.mkv.

Вставьте это в MTA/coreconfig.xml (menu_options всегда должен быть равен 7).

Запустите MTA_SA_unix.exe.

Getting MTA: SA to work with wine

You will need the latest version of Wine.

Set the MTA: SA, it does not matter what your path to the MTA and GTA: SA.

You need Wine version 0.9.49 or higher to run the MTA: SA.

Once install it, download the loader for * nix, and take it to your MTA San Andreas.

Download verdana, tahoma and tahomabd needed for CEGUI, and copy them to the $ HOME / .wine / drive_c / windows / fonts /

Open winecfg, Go to the "Graphics", and uncheck "Allow Pixel Shaders."

Remove from your folder file MTA San Andreas MTA / data / menu.mkv.

Put this in the MTA / coreconfig.xml (menu_options must always be equal to 7).

Start MTA_SA_unix.exe.
