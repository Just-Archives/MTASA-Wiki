Требуется пример Функция RU/xmlNodeSetName требует скриптового примера. Помогите, написав один.
Перед тем, как отправить изменения, прочтите Руководство по редактированию скриптовых примеров.

Устанавливает имя указанному XML узлу.

Синтаксис

bool xmlNodeSetName ( xmlnode node, string name )

Синтаксис ООП Помогите! Я не понимаю, что это!

Метод: xmlnode:setName(...)
Переменная: .name
Парная функция: xmlNodeGetName

Обязательные параметры

  • node: XML узел имя которого меняете.
  • name: новое имя, которое устанавливаете.

Возвращает

Возвращает true в случае успеха, false в противном случае.

Обязательные параметры

  • node: XML узел из которого получаем аттрибуты.

Возвращает

В случае успеха, возвращает таблицу с ключами, в виде соответствующих названиям аттрибутов, и с их значениями. Если узел не имеет аттрибутов, возвращает пустую таблицу. В случае неправильного регистра, возвращает false.
