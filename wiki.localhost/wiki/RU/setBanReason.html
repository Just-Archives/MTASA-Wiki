RU/setBanReason

From Multi Theft Auto: Wiki
Jump to: navigation, search

Эта функция устанавливает новую причину для бана.

Синтаксис

bool setBanReason( ban theBan, string theReason )

Обязательные параметры

  • theBan: Бан, для которого вы хотите поменять причину
  • theReason: новая причина (максимум 60 символов).

Возвращает

Возвращает true если новая причина установлена удачно, в противном случае false.

Пример

Этот пример добавляет команду setReason, используя которую изменяется причина по имени забаненного игрока. 

/setreason никнейм причина

function setReason (player,cmd,name,...)
	local reason = table.concat({...}," ")
	if name and reason then
		local bans = getBans()
		for i,v in ipairs(bans)do
			if getBanNick(v) == name then
				setBanReason(v,reason)
				outputChatBox("Удачно изменена причина бана.",player,0,125,0)
			end
		end
	end
end
addCommandHandler("setreason", setReason)
