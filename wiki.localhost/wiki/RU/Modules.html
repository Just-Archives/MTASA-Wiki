Модули

(Redirected from RU/Modules)
Jump to:navigation, search

Модули - расширения для Multi Theft Auto ядра Lua, предоставляющие новые внешние функции, которые недоступны в собственном ядре. Компилируются в библиотеку DLL или SO. Все указанные здесь модули не включаются в дистрибутивы MTA и должны быть вручную установлены.

Список модулей

Вы можете просмотреть список всех задокументированных в нашей вики модулей здесь.

SDK для модулей

Чтобы создать свой собственный модуль для MTA, вам необходимо воспользоваться Modules SDK. Вы можете скачать его по одной из ссылок ниже - выберите ссылку в соответствии с вашей операционной системой.

  • Linux
    • Официальный
      • (Удалите файлы с расширением .d, и добавьте #include <cstring> в Common.h)
        • (На 64-бит добавьте -m32 к строкам CPPFLAGS и LDFLAGS в Makefile)
  • Microsoft Windows
    • Зеркало (Mediafire)
    • Зеркало (MEGA)
