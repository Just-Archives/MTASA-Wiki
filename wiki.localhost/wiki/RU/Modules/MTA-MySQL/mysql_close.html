RU/Modules/MTA-MySQL/mysql close

From Multi Theft Auto: Wiki
Jump to:navigation, search


Эта функция предоставлена внешним модулем MTA MySQL. Вы должны установить этот модуль, чтобы использовать эту функцию.

Закрывает установленное соединение с MySQL-сервером.

Синтаксис

mysql_close(MySQLConnection handler)

Обязательные аргументы

  • handler. Действительная MySQL-ссылка.

Возвращаемые значения

Не возвращает никакого значения.

Пример

Пример 1. В этом примере создаётся соединение с MySQL-сервером и сразу же оно закрывается. 

handler = mysql_connect("localhost", "username", "password", "mta_users")

if not handler then
    outputDebugString("Не удалось подключиться к MySQL-серверу.")
else
    outputDebugString("Удалось подключиться к MySQL-серверу.")
    mysql_close(handler)
    outputDebugString("Соединение с MySQL-сервером закрыто.")
end
