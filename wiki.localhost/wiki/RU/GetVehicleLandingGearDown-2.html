RU/GetVehicleLandingGearDown

From Multi Theft Auto: Wiki
Jump to:navigation, search

Функция используется для проверки состояния шасси самолёта.

Синтаксис

bool getVehicleLandingGearDown ( vehicle theVehicle )

Необходимые аргументы

  • theVehicle: Транспортное средство для проверки шасси

Возвращения

Возвращает true если шасси опущены, false если шасси подняты.
Возвращает nil если транспортное средство не имеет шасси, или оно недействительно.

Пример

Эта функция говорит вам, поднять шасси, если вы находитесь в Hydra с опущенными шасси. 

function checkGear( thePlayer )
    local theVehicle = getPlayerOccupiedVehicle( thePlayer )
    if ( getElementModel(theVehicle) == 520 and getVehicleLandingGearDown( theVehicle ) == true ) then
        outputChatBox( "Поднимите шасси!", thePlayer )
    end
end
