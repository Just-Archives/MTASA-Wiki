RU/playSoundFrontEnd

From Multi Theft Auto: Wiki
Jump to: navigation, search

Эта функция воспроизводит внешний звук для указанного игрока.

[[{{{image}}}|link=]] Примечание: Эти звуки могут быть отключены с помощью setWorldSoundEnabled, поэтому, если эти звуки перестают работать, вы должны проверить, отключили ли вы звуки.

Синтаксис

Click to collapse [-]
Сервер
bool playSoundFrontEnd ( player thePlayer, int sound )

OOP Syntax Help! I don't understand this!

Method: player:playSoundFrontEnd(...)

Обязательные аргументы

  • thePlayer: Игрок, для которого вы хотите воспроизвести звук.
  • sound: Определяющий звуковой идентификатор для воспроизведения. Допустимые значения:
    • 0 - 20 Звуки выбора
    • 27 - 30 Звуки пуль (близко пролетающих)
    • 32 - 33 Звуки выбора
    • 34 Звук статичного радио (как при переключении станции)
    • 35 Звук остановки звука статичного радио
    • 37 - 38 Щелчок
    • 40 Звуки выбора
    • 41 - 42 Щелчок (когда нет патронов)
    • 43 - 45 Звук гоночного отсчета
    • 46 Починка
    • 47 Звук белого шума
    • 48 Звук остановки звука белого шума
    • 49 Краткий звук белого шума
    • 101 Отсчет/выбор
Click to expand [+]
Клиент
bool playSoundFrontEnd ( int sound )

OOP Syntax Help! I don't understand this!

Method: Sound.playFrontEnd(...)

Обязательные аргументы

  • sound: Определяющий звуковой идентификатор для воспроизведения. Допустимые значения:
    • 0 - 20 Selection sounds
    • 27 - 30 Bullet sounds
    • 32 - 33 Selection sounds
    • 34 Radio static
    • 35 Stop Radio static
    • 37 - 38 Tick
    • 40 Selection sounds
    • 41 - 42 Tick (no ammo)
    • 43 - 45 Race countdown
    • 46 Repair
    • 47 White noise static
    • 48 Stop White noise static
    • 49 Static short
    • 101 Countdown/selection

Возвращает

Возвращает «true», если звук был успешно воспроизведен, «false» в противном случае.

Пример

Click to collapse [-]
Сервер

Этот пример воспроизводит звук, когда появляется игрок. 

function onPlayerSpawn ( theSpawnpoint, theTeam )
    playSoundFrontEnd ( source, 16 )
end
addEventHandler ( "onPlayerSpawn", root, onPlayerSpawn )
Click to collapse [-]
Клиент

Этот пример воспроизводит звук, когда игрок вводит команду '/sound'. 

function onSoundEvent ( )
    playSoundFrontEnd ( 16 )
end
addCommandHandler("sound", onSoundEvent)
