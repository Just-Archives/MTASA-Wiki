Principales Cambios

  • Añadidas Sirenas para vehiculos
  • Arreglada descarga muy lenta de archivos map
  • Arreglados timeouts en cambios de mapa
  • Arreglados varios problemas, crashes y congelamientos
  • Actualizado la mayor cantidad de jugadores a 4096
  • Añadidos BASS efectos
  • Añadido Analog Control States
  • Añadido bullet sync
  • Arreglados muchos problemas al remplazar custom models
  • Soporte para Windows 8 (ambos 32-bits y 64-bits)
  • Abilidad de crear peds sin armas a travez de creacion de armas
  • Mejorada la estabilidad del map editor y añadidas nuevas caracteristicas
  • Instaladores para regular builds y nightlies estan ahora digitalmente registrados.

Cambios/Arreglos (Traduccion 50%)

  • Arreglado 'no estas permitido a ingresar al vehiculo' bug
  • Añadida prioridad de order a addEventHandler
  • Arreglada la variante 'engineless' a la NRG-500
  • Añadida la opcion para saltar la seleccion de Dual Monitor Resolution
  • Hecho playSound3D usar la posicion de la camara en lugar de la posicion del jugador cuando se determina la distancia
  • Añadidas opciones de color coding y sub-pixel positioning a dxDrawText
  • Añadida la habilidad de crear y modificar cubemaps y volume textures - Detalles: dxCreateTexture
  • Añadido un sistema de cache de modelos para reducir los retrasos de carga
  • Añadido un sistema de cache de ropa para CJ para reducir los congelamientos en el juego
  • Aumentada velocidad de los sistemas para llamar cliente y servidor
  • Añadida la opcion element a engineApplyShaderToWorldTexture
  • Añadido un bool opcional a getElementMatrix ( element, bool )
  • Añadido unrar para un procedimiento de actualizacion mas suave
  • Arreglado errores de sustitución de modelos personalizados, algunas veces con armas y pickups de armas.
  • Arreglado modelos personalizados de mejoras para vehículos no se muestran inmediatamente
  • Arreglado precisión de punto de golpe en onClientPlayerWeaponFire y añadido parámetro origen de tiro
  • Arreglado setPedAimTarget toda la dirección es incorrecta
  • Arreglado problema cuando las balas de los peds' se originan de una posición incorrecta
  • Arreglado modelos personalizados no se renderizan correctamente algunas veces parámetro
  • Arreglado distancia de un modelo LOD personalizado no se reinicia después de quitar
  • Añadido límite de tamaño para los archivos clientscript.log
  • Añadido la capacidad de disparar con cualquier arma con jetpack
  • Arreglado hacer temporizadores con menos errores
  • Arreglado algunos crashes de animaciones
  • Fixed getPedMoveState returns false when moving in crouch state
  • Fixed a bug when a resource that replace an object model (dff) and texture (txd) is stopped the object texture get white
  • Added ped pixel shaders
  • Added ability to layer multiple shaders on a world texture
  • Fixed Windows 8 missing dll error
  • Fixed setElementPosition for vehicles on a non streamed in position will make the vehicle spin very quickly
  • Added check for GTA file loading failures
  • Fixed map editor crash
  • Fixed floating vehicles when using setVehicleIdleRespawnDelay
  • Fixed showhud not fully working before the player has spawned
  • Fixed connect problem when using a domain name that starts with a number
  • Fixed missing font error message
  • Fixed getSoundLength returns 0 for sound streams (not radio streams)
  • Fixed a crash when taking a screenshot and minimizing then restoring
  • Fixed setElementFrozen killing players from falls
  • Fixed textures disappearing and flickering at certain camera angles

Bugs Arreglados

  • Se ha corregido colisiones de vehículos personalizados las cuales no cargaban
  • Se ha corregido que algunos modelos personalizados no cargaban
  • onVehicleExplode es triggeable para los coches RC tambien ahora.
  • Los proyectiles disparan a su distancia
  • Arreglada el tatuaje de araña de CJ
  • Speed up in entity collision detection client side
  • Fixed the glitchy GTASA animation when you block+sprint (where hands are repeatedly raised and lowered)
  • Fixed Objects scaled with setObjectScale aren't rendered when the unscaled bounding box goes off-screen
  • Fixed Always falling off bikes and motorbikes
  • Fixed Some vehicles become indestructible when a certain door is open
  • Possible fix for client freezing/crashing after playing a long while
  • Los marcadores en interiores ahora aparecen
  • Chineese/Japanese/Korean input fixed
