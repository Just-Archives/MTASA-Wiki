ES/onClientVehicleEnter

From Multi Theft Auto: Wiki
Jump to: navigation, search

Este evento se activa cuando un jugador entra a un vehículo.

Parametros

player elJugador, int asiento
  • elJugador: el jugador que entró al vehículo
  • asiento: el número de asiento donde el jugador esta sentado. 0 = conductor, los números más altos son de pasajeros.

Origen

La variable source de este evento corresponde al vehículo donde entró el jugador.

Ejemplo

En este ejemplo se actualiza una etiqueta GUI con el nombre del vehículo en el cual está el jugador. 

lblVehicle = guiCreateLabel(10, 200, 150, 20, "Actualmente a pie", false)
addEventHandler("onClientVehicleEnter", getRootElement(),
    function(elJugador, asiento)
        if elJugador == getLocalPlayer() then
            guiSetText(lblVehicle, "Actualmente en un " .. getVehicleName(source))
        end
    end
)
addEventHandler("OnClientVehicleExit", getRootElement(),
    function(elJugador, asiento)
        if elJugador == getLocalPlayer() then
            guiSetText(lblVehicle, "Actualmente a pie")
        end
    end
)
