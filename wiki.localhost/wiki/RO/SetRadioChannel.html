RO/SetRadioChannel

From Multi Theft Auto: Wiki
Jump to:navigation, search

Această funcție setează stația radio curentă, chiar și atunci când nu ești într-un vehicul.

[[{{{image}}}|link=]] Observație: Uneori această funcție nu funcționează atunci când stația radio este schimbată din motive necunoscute. Dacă întâmpinați această problemă, adăugați setRadiOChannel(0) la începutul script-ului, în afara oricărei alte funcții.

Sintaxă

bool setRadioChannel (int ID)

Argumente Necesare

  • ID: ID-ul stației radio pe care doriți să o redați.
  • 0: Radio Oprit
  • 1: Playback FM
  • 2: K-Rose
  • 3: K-DST
  • 4: Bounce FM
  • 5: SF-UR
  • 6: Radio Los Santos
  • 7: Radio X
  • 8: CSR 103.9
  • 9: K-Jah West
  • 10: Master Sounds 98.3
  • 11: WCTR
  • 12: User Track Player

Redare

Redă true dacă stația radio a fost setată cu succes. În caz contrar, false.

Exemplu

Acest exemplu adaugă comanda setradio care permite schimbarea stației radio curente prin introducerea noului ID.

Click to collapse [-]
Client
addCommandHandler("setradio",
    function (command, stationID)
        local result = setRadioChannel(tonumber(stationID))
        if (result) then -- Verificăm dacă ID-ul introdus este unul valid;
            outputChatBox("Stația radio a fost schimbată la ID: " .. getRadioChannelName(tonumber(stationID)) .. "!" )
        else
            outputChatBox("ID stație radio nevalid. ID-urile valide sunt cuprinse între 0-12." )
        end
    end
)

Probleme

ID Problemă Descriere
#423 Uneori setRadioChannel nu funcționează atunci când nu ești într-un vehicul.
