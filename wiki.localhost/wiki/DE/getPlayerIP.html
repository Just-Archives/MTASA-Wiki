DE/getPlayerIP

From Multi Theft Auto: Wiki
Jump to:navigation, search

Diese Funktion gibt einen string aus, der die IP-Adresse des Spieler's enthält.

Syntax

string getPlayerIP ( player thePlayer )

Erfordrliche Argumente

  • thePlayer: Der Spieler von dem die IP-Adresse ausgegeben werden soll.

Ausgabe

Gibt einen string mit der IP-Adresse des Spieler's zurück.

Beispiel

Dieses Beispiel gibt die IP-Adresse des player's im Chat aus. 

function printIp ( thePlayer )
	outputChatBox ( "IP: " .. getPlayerIP ( thePlayer ), thePlayer )
end
addCommandHandler ( "ip", printIp )
