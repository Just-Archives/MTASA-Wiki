Giriş

Bu sayfa GTA:SA'daki araçların ID listesini içermektedir, ayrıca vehicles.ide dosyasının içinde de listelenmiştir.Bu araç ID numaraları bazı araç script fonksiyonlarında kullanılmaktadır.
