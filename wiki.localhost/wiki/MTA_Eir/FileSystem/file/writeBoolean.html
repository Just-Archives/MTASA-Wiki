MTA:Eir/FileSystem/file/writeBoolean

From Multi Theft Auto: Wiki
Jump to: navigation, search

This function attempts to write a boolean into a file/stream and returns how many bytes have actually been written. The amount of bytes written should be one.

Syntax

int file.writeBoolean ( boolean value )

Arguments

  • value: a boolean value to write into the file/stream

Returns

Returns the amount of bytes written during the write operation. Returns false if value is not a valid boolean type
