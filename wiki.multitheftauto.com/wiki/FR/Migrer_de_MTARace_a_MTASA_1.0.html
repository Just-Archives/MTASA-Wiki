FR/Migrer de MTARace a MTASA 1.0.x

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

MTA:Race was the first Multi Theft Auto release for Grand Theft Auto San Andreas. Since its release, we've seen thousands of maps and a vast, active community develop and mature. Despite this, for everything there is a time to move on. Multi Theft Auto: San Andreas 1.0, in conjunction with the latest version of the Resource:Race gamemode provides a stable platform for server admins and gamers alike to upgrade to the most powerful GTA Race experience yet.

Pourquoi mettre à jour ?

Les versions récentes vous offres une expérience de jeu plus confortable.

Scripting

Avec l'arrivée de MTA:SA 1.0, arrive la révolution des scripts dans le monde de MTA. Les apports sont nombreux notamment grâce à l'interface LUA.

  • Intégration complète des scripts sur vos serveurs. Administration en jeu et non externe (via la console)
  • Profitez des avantages liés aux fonction lua pour MTA Dévellopez vos scripts, les possibilités sont illimitées. Un exemple concret de cette mise à jour est "The Traffic Sensor", qui utilise vous indique qu'un vehicule est proche de vous ou derrière, on trouve aussi un Top Times sur les courses, qui affiche le classement des joueurs les plus rapides sur le circuit courant.
  • Personnaliser vos maps grâce au LUA. En liant un script LUA à celle-ci, vous pouvez contrôler une multitude de paramètres. Textures, des objets qui bougent, des explosions ainsi que modifier la gravité.
  • Entièrement personnalisable. Personnaliser votre serveur, ajouter des images à votre serveur, des objets, des sons, des textures pour les véhicules, ajouter votre logo lorsque l'on rejoins votre serveur etc.

Gameplay

La version MTA:SA 1.0 offre plus de possibilités et fait donc une version plus complète que MTA:Race. Le point le plus important est sans doute le mode race qui se voit évoluer d'une centaine de nouveautés pour toujours plus de fun.

  • Max 128 joueurs sur un serveur race. C'est bien plus que 32. Vous vous demandez "Comment 128 joueurs peuvent-ils jouer simultanément avec fluidité ?". Aucun soucis, La dernière version offre ce confort.

Un mode important fait son apparition : Le ghostmode pour passer au travers de ses concurrents.

  • MTA 1.0 est supporter par les nouveaux systèmes d'exploitation de Windows en 32 ou 64 bits.

C'est grâce à l'accès libre au code source de MTA:SA 1.0, que la comptabilités est rendue possible.

  • Une synchronisation améliorée. En passant en 1.0, la première chose que les joueurs vont remarquer c'est cette synchronisation performante qui n'existait pas avant. Précision et fluidité sont les maîtres mots de cette version.
  • L'ajout de l'addon "Traffic Sensor" permet ou joueur de savoir où se trouve leur concurrents en pleine course. cela vous permet d'anticiper les mouvement de votre adversaire.
  • Le tout nouveau Map Editor en open source signifie que vous pouvez dès lors créer vos maps plus facilement.
  • MTA:SA 1.0 offre un mode de qualité pour tout ses joueurs de plus en plus nombreux.

Le Paradis des Propriétaires de Serveur

Avec cette dernière version, les propriétaires de serveurs on le contrôle total.Vous êtes sur le point de customisé votre serveur pour offrir une nouvelle expérience de jeu afin de devenir un serveur réputé et unique.

  • Améliorations en jeu pour les Admins. MTA:SA 1.0 vous offre un panel d'administration complet où vous règlerez tout en quelques clics.
  • Meilleure stabilité, performance et bande passante. La version 1.0 à été soumise à de nombreux tests et les développeurs ont visé bien plus haut que sur MTA:Race. Une bande passante augmentée, ce qui fait le bonheur des serveurs MTA:SA.
  • Limite de FPS. Nous l'avons tous vécu - la frustration de la limite du FPS sur le MTA:Race. L'arrivée de la version 1.0 signifie que les propriétaires de serveur peuvent eux même définir cette limite.
  • Personnalisable. Comme répété plusieurs fois - tout est sous votre contrôle. En utilisant le LUA vous ouvrez une infinité de créations pour améliorer l'expérience de jeu sur vos serveurs. Le nouveau système de plugins vous aide à ajouter ou supprimer les plugins facilement.

Communauté

Si vous n'avez pas trouver la solution à un de vos problèmes sur MTA, nous seront heureux de vous aider sur le forum. Le nombre grandissant de scripter pour MTA:SA 1.0 signifie aussi que c'est le début d'une nouvelle ère pour MTA et d'une communauté toujours plus nombreuse que sur MTA:Race.

Comment dois-je mettre à jour ?

Joueurs

Pour mettre à jour votre version rien de plus simple. Vous avez tout à y gagner. MTA:SA 1.0 peut être installer sans rentrer en conflit avec l'installation de MTA:Race, ce dernier fonctionnera toujours.


La seule chose que vous aillez à faire est d'aller sur download page Téléchargez MTA:SA 1.0.

Propriétaires

Il est très simple de mettre à jour votre serveur. Les maps de votre serveur MTA:Race ne sont pas compatibles avec cette nouvelle version. Installer simplement le dernier set pour serveur, et utiliser le convertisseur de maps Maps Converter pour les convertirent en versions compatibles MTA:SA 1.0. A partir de ce moment vos maps tournent avec race gamemode et peuvent être administrées via map manager et le vote manager.
