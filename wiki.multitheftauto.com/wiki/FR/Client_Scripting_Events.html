Cette page liste toutes les événements disponibles côté-client et utilisables dans le "Deathmatch mod".

Pour demander une nouvelle fonction ou un événement, utilisez ce lien.

Les événements côté-serveur se trouvent ici

La description indique à quel moment l'événement est appelé, cliquer pour plus d'informations.

Zone de collision (colshape)

  • onClientColShapeHit (Quand un élément entre dans une zone de collision)
  • onClientColShapeLeave (Quand un élément sort d'une zone de collision)


Eléments

  • onClientElementDestroy (Quand un élément est détruit)
  • onClientElementColShapeHit (Quand un joueur/vehicule entre dans une zone de collision)
  • onClientElementColShapeLeave (Quand un joueur/vehicule sort d'une zone de collision)
  • onClientElementDataChange (Quand une donnée d'un élément est modifiée)
  • onClientElementStreamIn (Quand un élément est streamer pour le client)
  • onClientElementStreamOut (Quand un élément arrête d'être streamer pour le client)

GUI (Interface)

  • onClientGUIClick (Quand un élément d'interface est cliqué)
  • onClientGUIDoubleClick (Quand un élément d'interface est double cliqué)
  • onClientGUIChanged (Quand une textbox est modifiée)
  • onClientGUIAccepted (Quand la toucher entré ou le bouton validé est pressé)
  • onClientGUIScroll (Quand la barre de scroll est utilisée)
  • onClientClick (Quand le client clique sur un élément du jeu)
  • onClientDoubleClick (Quand le client double clique sur un élément du jeu)
  • onClientKey (Quand le client presse une touche du clavier)
  • onClientCharacter (Quand le client appuie sur un charactere alphanumérique)
  • onClientMouseEnter (Quand la souris entre au dessus d'une interface)
  • onClientMouseLeave (Quand la souris quitte une interface)
  • onClientMouseMove (Quand la souris se déplace au dessus d'une interface)
  • onClientMouseWheel (Quand la molette de la souris est utilisé au dessus d'une interface)
  • onClientCursorMove (Quand la souris est déplacée)
  • onClientGUIMouseDown
  • onClientGUIMouseUp
  • onClientGUIMove (Quand un élément d'interface est déplacé)
  • onClientGUISize (Quand un élément d'interface est redimensionné)
  • onClientGUITabSwitched (Quand l'onglet d'une interface change)
  • onClientGUIComboBoxAccepted (Quand un item d'une liste a été choisi)
  • onClientGUIFocus (Quand un élément d'interface gagne le focus. Ex: Clique sur une zone de texte)
  • onClientGUIBlur (Quand une élément d'interface perd le focus)


Marqueurs

  • onClientMarkerHit (Quand le joueur entre sur un marqueur)
  • onClientMarkerLeave (Quand le joueur quitte un marqueur)


Piétons

  • onClientPedDamage (Quand un piéton subit des dégats)
  • onClientPedWasted (Quand un piéton est tué ou meurt)
  • onClientPedWeaponFire (Quand un piéton utilise une arme)
  • onClientPedChoke (Event non utilisable)
  • onClientPedHeliKilled (Quand un piéton est tué par un hélico)
  • onClientPedHitByWaterCannon (Quand un piéton est tué par un canon à eau)


Pickups

  • onClientPickupHit (Quand un élément touche un pickup)
  • onClientPickupLeave (Quand un élément quitte un pickup)
