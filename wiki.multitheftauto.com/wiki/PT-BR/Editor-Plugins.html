Introdução

O editor oferece funções e comandos para interagir com recursos externos. Normalmente ao criar um elemento ou realizar ações fora do editor não permitem que isso possa ser gerenciado por ele. É como se o Editor fosse cego e o elemento ali criado não fosse visto por ele - portanto, não clicável. Entretanto, ao importar elementos, o editor os fazem compatíveis com ele e assim poderão ser manipulados pelo usuário através de sua interface - além de salva-los no arquivo .map. Na prática isso significa desenvolvimento de plugins ou extensões.

Comandos

O comando "import" foi criado pelo editor para permitir usuários importarem elementos de um recurso especificado: 

import <nome-do-recurso>
  • nome-do-recurso: É o nome do qual você deseja importar.

Assim, os elementos do recurso serão importados. Um exemplo disso seria importar modelos customizados. Enquanto o editor não é capaz de realizar esta ação sozinho, seu recurso pode fazer isso por ele, e importar as coordenadas desse objeto - criando um mapa mais customizado ainda. Um exemplo prático abaixo seria fazer isso com o mapa sth-aztec:

  • Inicie o editor
  • Inicie (mas não abra no Editor) o sth-aztec. Isso irá iniciar o mapa e carregar automaticamente os modelos customizados, pois no script dele já contém as instruções para isso.
  • Digite "import sth-aztec" e os objetos do mapa serão importados ao editor
  • Agora o mapa com os objetos customizados carregarão
