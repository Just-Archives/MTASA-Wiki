DE/setRadioChannel

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Diese Funktion setzt den derzeitigen Radio Kanal, auch wenn man nicht im Fahrzeug ist.

[[{{{image}}}|link=|]] Bemerkung: Diese Funktion funktioniert scheinbar manchmal nicht aus unerklärlichen Gründen beim Ändern eines anderen Kanals. Wenn du dieses Problem feststellst, füge setRadioChannel(0) am Anfang deines Skriptes hinzu.

Syntax

bool setRadioChannel ( int ID )

Benötigte Argumente

  • ID: Die ID der Radiostation, die du spielen möchtest.
  • 0: Radio Off
  • 1: Playback FM
  • 2: K-Rose
  • 3: K-DST
  • 4: Bounce FM
  • 5: SF-UR
  • 6: Radio Los Santos
  • 7: Radio X
  • 8: CSR 103.9
  • 9: K-Jah West
  • 10: Master Sounds 98.3
  • 11: WCTR
  • 12: User Track Player

Rückgabe

Gibt true zurück wenn der Kanal erfolgreich gesetzt wurde, false wenn nicht.

Beispiel

Dieses Beispiel erstellt den Command setradio, welcher die derzeitige Radiostation wechseln kann.

Click to collapse [-]
Client
addCommandHandler ( "setradio",
    function ( command, stationID )
        local result = setRadioChannel ( tonumber( stationID ) )
        if result then -- Prüfen es erfolgreich war (gültige ID)
            outputChatBox ( "Deine Radiostation wurde gewechselt zu " .. getRadioChannelName ( tonumber ( stationID ) ) .. "!" )
        else
            outputChatBox ( "Ungültige Radiostation ID, gültig sind 0-12." )
        end
    end
)
