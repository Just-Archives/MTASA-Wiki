DE/isVehicleFuelTankExplodable

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Diese Funktion prüft, ob der Tank eines Fahrzeugs explodieren kann.

Syntax

bool isVehicleFuelTankExplodable ( vehicle theVehicle )

Benötigte Argumente

  • theVehicle Das Fahrzeug, von dem der Status abgefragt werden soll.

Return

Gibt true zurück, wenn das Fahrzeug richtig ist und der Tank explodieren kann. false andererseits.

Beispiel

In diesem Beispiel wird überprüft, ob der Tank explodieren kann und dann in der Chatbox ausgegeben. 

[lua,n]function checkTank ( )
    local veh = getPedOccupiedVehicle(source)
    if isVehicleFuelTankExplodable(veh) then
         outputChatBox("Das Fahrzeug kann explodieren")
    elseif not isVehicleFuelTankExplodable(veh) then
         outputChatBox("Das Fahrzeug kann nicht explodieren")
    end
end
addCommandHandler("check", checkTank)
