DE/addVehicleUpgrade

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Mit dieser Funktion lassen sich Tuningteile in einem Auto einbauen.

Syntax

bool addVehicleUpgrade ( vehicle theVehicle, int upgrade )

Benötigte Argumente

  • theVehicle: Ein Fahrzeug Element
  • upgrade: Die Upgrade-ID des Tuningteils, das eingebaut werden soll. Eine Liste findet man hier: http://wiki.multitheftauto.com/wiki/Vehicle_Upgrades

Returns

Gibt ein true zurück, wenn das Upgrade erfolgreich eingebaut wurde, false wenn nicht.
