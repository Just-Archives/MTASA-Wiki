CustomLabel:removeElement

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

This function removing object from Label. Removing all parameters. Recommend after this function removing object with destroyElement or CustomWidget:destroy method.

Syntax

nil CustomLabel:removeElement(cwidget Element)

Required Arguments

  • Element - Custom Widget element, what connected to Label.

Example

This example creates window with scroll pane with 2 buttons: 

local Window = CustomWindow.create(50, 50, 190, 100, "Example")
local Label = CustomLabel.create(5, 25, 180, 70, "Example", false, Window)

local Button = CustomButton.create(50, 5, 80, 25, "Button", false, Label)

Label:removeElement(Button)
Button:destroy()
