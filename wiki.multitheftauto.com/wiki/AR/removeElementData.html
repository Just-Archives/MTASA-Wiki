AR/removeElementData

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

هذه الوظيفة تقوم بإزالة البيانات للعنصر بواسطة المفتاح المعطى.

تركيب الجملة البرمجية

bool removeElementData ( element theElement, string key )

OOP Syntax Help! I don't understand this!

Method: element:removeData(...)

العناصر المطلوبة

  • theElement: العنصر الذي تود في إزالة البيانات منه.
  • key: مفتاح البيانات الذي تود إزالته.

النواتج

يقوم بإرجاع صحيح إذا تمت إزالة البيانات بنجاح، خطأ إذا كان المفتاح المحدد غير موجود لدى العنصر أو العنصر غير صالح.

مثال

سيتم اعطاء اللاعب بيانات عندما يرسبن وسيتم إزالتها عندما يموت. 

function spawn()
    setElementData(source,"spawned",true)
end
addEventHandler ( "onPlayerSpawn", getRootElement(), spawn )
function wasted()
    removeElementData(source,"spawned")
end
addEventHandler ( "onPlayerWasted", getRootElement(), wasted )
