ES/Funciones de Servidor

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Esta página muestra todas las funciones de servidor que han sido implementadas y están disponibles como funciones nativas. Para pedir que se cree alguna función o un evento, visita el artículo oficial de Pedido de Funciones y Eventos o https://github.com/multitheftauto/mtasa-blue/issues.

Ten en cuenta que las funciones se pueden también extender cargando un modulo dinámico que entregue una nueva funcionalidad, como la base de datos MySQL. Estas funciones no son nativas y requieren que el módulo esté cargado para funcionar.

Véase Módulos para una lista de funciones de servidor no-nativas y módulos que están disponibles.

Para las funciones de utilidad no-nativas hechas por usuarios de MTA, revisa la página de Funciones Útiles.

Las funciones de cliente se pueden encontrar en el artículo sobre Funciones de cliente.

También ten en cuenta que casi todas las funciones presentes no presentan una traducción en su definición
