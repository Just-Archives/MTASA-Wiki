ES/CreatePickup

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Esta función crea un elemento pickup, que se coloca en el mundo GTA y se puede recoger para recuperar salud, armadura o un arma.

Syntaxis

pickup createPickup ( float x, float y, float z, int Tipo, int cantidad/arma/modelo, [ int respawnTime = 30000, int municion = 50 ] )

Argumentos Requeridos

  • x: La posición x del Pickup, en coordenadas del mundo.
  • y: La posición y del Pickup, en coordenadas del mundo.
  • z: La posición z del Pickup, en coordenadas del mundo.
  • Tipo: Esto representa el Tipo del Pickup, representando los siguientes tipos:
    • 0: Pickup de Vida
    • 1: Pickup de Armadura
    • 2: Pickup de Armas
    • 3: Pickup Personalizado
  • cantidad:: Esto Representa la Cantidad de puntos de Vida o Armadura que el Pickup Dara.

O

  • arma: Si el Tipo de pickup es de Arma, Entonces lo representa el weapon ID del pickup de Arma. Cuando es usado con el Pickup de arma, El Parámetro de munición puede ser usado.

O

  • modelo: Si el pickup es un Modelo Personalizado, esta es la id del modelo a utilizar. Muchos modelos que no son pickup pueden ser utilizados, aunque algunos pueden causar accidentes. La siguiente lista es una lista de modelos diseñados para ser utilizados como Pickup:
    • 1212: Money (Fajo de Billetes)
    • 1240: Vida (Corazón)
    • 1242: Armadura
    • 1239: Info icon
    • 1272: Casa (Azul)
    • 1273: Casa (Verde)
    • 1274: Dinero (Simbolo de Dollar)
    • 1241: Adrenaline
    • 1247: Bribe
    • 1248: Logo GTA III
    • 1252: Bomba del GTA III
    • 1253: Photo op
    • 1254: Cráneo
    • 1274: Icono de Dinero
    • 1275: Camiseta Azul
    • 1277: Caset de Guardado
    • 1313: 2 Cráneos
    • 1276: Estatua Tiki
    • 1310: Paracaidas (with leg straps)
    • 1318: Down arrow
    • 1279: Droga

Tambien Puede usarse Cualquier ID De Otro objeto-

Argumentos Opcionales

  • respawnTime: Tiempo en el que Aparecerá de Nuevo el Pickup
  • municion: Esto Representa la Cantidad de munición que el Pickup Tendrá. Esto Solo es Válido si el Pickup es de un Arma.

Returns

Retorna un pickup si fue creado satisfactoriamente, false de otra manera.

Example

Server

Este ejemplo crea un pickup después que el jugador muera de modo que él deje caer su arma. 

function crearPickup ( totalammo, killer, killerweapon, bodypart ) --Cuando el jugador muere
    x, y, z = getElementPosition ( source ) --Obtener la posición de la persona que murió y definirlo como x, y, z
    armaactual = getPlayerWeapon ( source ) --Obtener el arma actual de la persona muerta
    createPickup ( x, y, z, 3, armaactual, 10000, totalammo )
end
addEventHandler ( "onPlayerWasted", getRootElement(), crearPickup ) --Añadir el evento onPlayerWasted
