RO/resizeBrowser

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Permite redimensionarea navigatoarelor web CEF la runtime.


Atenționare: Nu folosiți această funcție alături de onClientRender deoarece recreează textura de la bază intern. (ceea ce este o operațiune costisitoare)

Sintaxă

bool resizeBrowser (browser webBrowser, float width, float height)

Argumente Necesare

  • webBrowser: Navigatorul web pe care doriți să îl redimensionați.
  • width: Noua lățime a navigatorului web.
  • height: Noua înălțime a navigatorului web.

Redare

Redă true dacă navigatorul web a fost redimensionat cu succes, false dacă a apărut o problemă.

Exemplu

Exemplu pentru redimensionarea navigatorului web prin comandă. 

screenWidth, screenHeight = guiGetScreenSize()
initBrowser = guiCreateBrowser(0, 0, screenWidth, screenHeight, true, true, false)

addCommandHandler("resize",
function (cmd, width, height)
    local browser = guiGetBrowser(initBrowser)
    local width, height = tonumber(width), tonumber(height)
    resizeBrowser( browser, width, height )
end)
