Category:Формы коллизии

From Multi Theft Auto: Wiki
(Redirected from RU/Element/Colshape)
Jump to navigation Jump to search

Класс COL shape представляет собой невидимые формы обнаружения столкновения, которые можно создать в мире GTA. Формы коллизии обычно используются для обнаружения движущихся через них физических объектов и выполнения действий, когда они это делают.

Тип элемента этого класса - colshape.

XML синтаксис

<colcube posX="1024.768" posY="1248.1024" posZ="800.600" width="100" height="100" depth="100"/>
<colsphere posX="1024.768" posY="1248.1024" posZ="800.600" radius="100"/>
<coltube posX="1024.768" posY="1248.1024" posZ="800.600" radius="30" height="15"/>
<colrectangle posX="1024.768" posY="1248.1024" posZ="800.600" width="100" depth="61.8"/>
<colcircle posX="1024.768" posY="1248.1024" posZ="800.600" radius="30"/>

Обязательные аргументы

  • posX: Поплавок, представляющий положение формы коллизии по оси X.
  • posY: Поплавок, представляющий положение формы коллизии по оси Y.
  • posZ: Поплавок, представляющий положение формы коллизии по оси Z.
  • radius: Радиус формы коллизии (только сферы, трубки и круги).
  • width: Ширина формы коллизии (только прямоугольники и кубики).
  • depth: Глубина формы коллизии (только прямоугольники и кубики).
  • height: Высота формы коллизии (только кубики).

Необязательные аргументы

  • dimension: Измерение, в котором будет находиться форма коллизии
