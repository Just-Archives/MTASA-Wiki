Класс Object представляет статические трехмерные модели в мире GTA. Объекты могут представлять только модели, созданные сервером, они не могут представлять модели, которые являются частью ландшафта GTA по умолчанию. Примеры объектов включают модели зданий, дороги и рельеф.

Тип элемента этого класса - "object".

XML синтаксис

<object model="" posX="" posY="" posZ="" rotX="" rotY="" rotZ="" interior="" dimension="" scale="" collisions="" alpha="" frozen="" />

Обязательные аргументы

  • model: ID создаваемого объекта. Поскольку в GTA есть тысячи объектов, их трудно задокументировать в вики. Вместо этого их можно найти с помощью обозревателя объектов в редакторе карт.
  • posX: Поплавок, представляющий положение объекта по оси X.
  • posY: Поплавок, представляющий положение объекта по оси Y.
  • posZ: Поплавок, представляющий положение объекта по оси Z.

Необязательные аргументы

  • rotX: Поплавок, представляющий вращение объекта по оси X в градусах.
  • rotY: Поплавок, представляющий вращение объекта по оси Y в градусах.
  • rotZ: Поплавок, представляющий вращение объекта по оси Z в градусах.
  • interior: Интерьер, в котором будет находиться объект.
  • dimension: Измерение, в котором будет находиться объект.
ДОБАВЛЕНО/ОБНОВЛЕНО В ВЕРСИИ 1.1.0 :
  • scale: Масштаб объекта.
  • collisions: Включение/отключение коллизии объекта.
  • alpha: Изменяет прозрачность объекта.
  • frozen: Устанавливает, должен ли объект быть заморожен (также известный как статический).
