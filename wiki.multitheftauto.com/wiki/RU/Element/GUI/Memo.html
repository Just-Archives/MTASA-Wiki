Пример GUI памятки.

Класс GUI memo представляет памятки GUI в системе графического интерфейса пользователя MTA. Это большие многострочные текстовые поля, в которые вы можете вводить текст по умолчанию. Рекомендуется использовать редактируемые поля для ввода меньшего текста.

Тип элемента этого класса - "gui-memo".
