Класс Marker представляет цветные трёхмерные фигуры в мире GTA. Имеются разные типы маркеров. В скриптах маркеры часто используются, чтобы отмечать места и запускать какое-то действие, когда игрок входит в них.

Тип элемента этого класса - "marker".

Размер маркера НЕ может быть изменен из XML файла и по умолчанию равен 4.0.

XML синтаксис

<marker posX="" posY="" posZ="" type="" .../>

Обязательные аргументы

  • posX: Поплавок, представляющий положение маркера по оси X.
  • posY: Поплавок, представляющий положение маркера по оси Y.
  • posZ: Поплавок, представляющий положение маркера по оси Z.

Необязательные аргументы

  • type: Визуальный тип маркера. Типы маркеров:
    • "checkpoint": Гоночный чекпоинт, очень высокий, но не бесконечный. В одиночной игре использовался для обозначения пути гонки.
    • "ring": Светящееся кольцо, обычно используется для самолётов.
    • "cylinder": Цилиндр, представляет собой подобие чекпоинта, но не такой высокий. В одиночной игре использовался для обозначения точек начала миссий.
    • "arrow": Стрелка, представляет собой перевёрнутый к земле конус. В одиночной игре использовался как вход в здания.
    • "corona": Светящийся цветной шар.
  • color: Цвет маркера в HTML формате "Пример: #RRGGBBAA". По умолчанию красный.
