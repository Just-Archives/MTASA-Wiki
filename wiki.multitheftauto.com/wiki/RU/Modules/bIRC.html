Базовый IRC модуль (bIRC) предоставляет полнофункциональный IRC echo для сервера MTA:SA. На данный момент доступен только для Windows серверов на MTA:SA 1.0 (другие не протестированы).

Установка

Распакуйте файл ml_birc.dll в папку server\mods\deathmatch\modules\ вашего сервера.

mtaserver.conf

Добавьте следующую строку в mtaserver.conf вашего сервера: 

<module src="ml_birc" />
