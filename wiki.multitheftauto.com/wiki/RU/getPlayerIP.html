RU/getPlayerIP

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Эта функция возвращает строку, содержащую IP адрес игрока.

Syntax

string getPlayerIP ( player thePlayer )

Синтаксис ООП Помогите! Я не понимаю, что это!

Метод: player:getIP(...)
Переменная: .ip

Обязательные аргументы

  • thePlayer: игрок, IP которого вы хотите получить.

Возвращает

  • Строку, содержащую IP указанного игрока.
  • false, если указанного игрока не существует.

Пример

Этот пример показывает IP игрока в чате. 

function printIp ( thePlayer )
	outputChatBox ( "IP: " .. getPlayerIP ( thePlayer ), thePlayer )
end
addCommandHandler ( "ip", printIp )
