RU/getHeliBladeCollisionsEnabled

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Эта функция получает состояние возможности колизий вертолетных лопастей конкретной машины.

Синтаксис

bool getHeliBladeCollisionsEnabled ( vehicle theVehicle )

Синтаксис ООП Помогите! Я не понимаю, что это!

Метод: vehicle:areHeliBladeCollisionsEnabled(...)
Переменная: .heliBladeCollisionsEnabled
Парная функция: setHeliBladeCollisionsEnabled

Обязательные аргументы

  • theVehicle: проверяемая машина.

Возвращает

Возвращает true если колизии разрешены для конкретной машины, false если колизии не разрешены для конкретной машины, она не является вертолетом или даны неправильные аргументы.

Примеры

Client

Этот пример показывает состояние колизий лопастей 

function onVehicleEnter ( seat, jacked )
	--If the player entered a helicopter
	if ( getVehicleType ( source ) == "Helicopter" ) then
		local state = getHeliBladeCollisionsEnabled‎(source )
		outputChatBox("Helicopter blades collisions are turned ".. (state and "on" or "off") )
	end
end
addEventHandler ( "onClientVehicleEnter", localPlayer, onVehicleEnter )
