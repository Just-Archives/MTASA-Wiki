RU/showCursor

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Эта функция используется для отображения или скрытия указателя игрока.

[[{{{image}}}|link=|]] Примечание: Независимо от состояния курсора, которое вы устанавливаете с помощью этой функции, курсор всегда будет отображаться, когда меню, строка ввода чата или консоль активны или если другой ресурс вызвал эту функцию.

Синтаксис

Click to collapse [-]
Сервер
bool showCursor ( player thePlayer, bool show, [ bool toggleControls = true ] )

Обязательные аргументы:

  • thePlayer: Игрок, которому вы хотите скрыть или показать курсор.
  • show: Логическое значение, определяющее, показать ли ( true ) или скрыть ( false ) курсор.

Настраиваемые аргументы

Примечание. При использовании необязательных аргументов Вам может потребоваться предоставить все аргументы перед тем аргументом, который Вы хотите использовать. Для более подробной информации ознакомьтесь с необязательными аргументами.

  • toggleControls: Логическое значение, определяющее, следует ли отключать элементы управления во время показа курсора. true подразумевает, что элементы управления отключены, false означает, что элементы управления остаются включенными.

Клиент
bool showCursor ( bool show, [ bool toggleControls = true ]  )

Обязательные аргументы

  • show: Логическое значение, определяющее, показать ли ( true ) или скрыть ( false ) курсор.

Настраиваемые аргументы

Примечание. При использовании необязательных аргументов Вам может потребоваться предоставить все аргументы перед тем аргументом, который Вы хотите использовать. Для более подробной информации ознакомьтесь с необязательными аргументами.

  • toggleControls: Логическое значение, определяющее, следует ли отключать элементы управления во время показа курсора. true подразумевает, что элементы управления отключены, false означает, что элементы управления остаются включенными.

Возвращает

Возвращает «true», если указатель игрока был показан или спрятан успешно, иначе «false».

Пример

Click to collapse [-]
Сервер

В этом примере показан курсор для игрока с именем «Dave», после чего выводится сообщение, если оно было успешно показано. 

local thePlayer = getPlayerFromName ( "Dave" )              -- Получаем игрока по имени «Dave» 
if thePlayer then                                           -- если мы его получили
    showCursor ( thePlayer, true )                          -- показываем ему курсор
    if isCursorShowing ( thePlayer ) then                   -- проверяем, был ли ему показан курсор?
        outputChatBox ( "Курсор показан игроку Dave" ) -- Выводим сообщение в чат
    end
end
Click to collapse [-]
Клиент

Этот пример показывает курсор все время 

showCursor ( true ) -- Показать курсор
showCursor ( false ) -- Скрыть курсор
