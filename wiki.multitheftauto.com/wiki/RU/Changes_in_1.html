Основные новшества

  • Добавлены функции для удаления и восстановления компонентов карты San Andreas

Клиент

Новые функции

  • Добавлена removeWorldModel (Доступна с 1.2.0-3591)
  • Добавлена restoreWorldModel (Доступна с 1.2.0-3591)
  • Добавлена restoreAllWorldModels (Доступна с 1.2.0-3591)
  • Добавлена getBirdsEnabled
  • Добавлена setBirdsEnabled
  • Добавлена setOcclusionsEnabled

Новые события

  • Добавлено onClientVehicleCollision

Изменения

  • В правом нижнем углу браузера серверов теперь отображается количество игроков онлайн
  • Новый CEGUI скин "Lighter Black" от Aibo


Исправления

  • Исправлен баг с незагружающимися новыми коллизиями для транспортных средств
  • Исправлен баг, когда новые текстуры моделей иногда не загружались
  • onVehicleExplode теперь работает и для RC транспортных средств
  • Снаряды стреляют с нормальной скоростью
  • Исправлен паукообразный CJ
  • Ускорено обнаружение entity-колизий на клиентской стороне
  • Исправлена кривая анимация GTASA, когда жмете блок+спринт (когда руки повторяясь поднимаются/опускаются)
  • Исправлен рендеринг объектов, отмасштабированных через setObjectScale, когда они выходили за пределы экрана
  • Исправлен баг постоянного падения с велосипедов и мотоциклов
  • Исправлен баг, когда некоторые транспортные средства становились неуничтожаемыми, если у них открыта определенная дверь
  • Возможно, исправление для зависаний/вылетов мультиплеера после продолжительной игры
  • Маркеры в интерьерах не появляются
  • Ввод с китайской/японской/корейской раскладок клавиатуры исправлен

Сервер

Новые функции

  • Добавлена takePlayerScreenShot
  • Добавлена setOcclusionsEnabled
  • Добавлена removeWorldModel (Доступна с 1.2.0-3624)
  • Добавлена restoreWorldModel (Доступна с 1.2.0-3624)
  • Добавлена restoreAllWorldModels (Доступна с 1.2.0-3624)

Новые события

  • Добавлено onPlayerScreenShot

Изменения

  • Пока никаких

Исправления

  • Исправлена застапаривающаяся закачка карт при больших объемах передачи

Ресурсы

  • Admin: добавлена опция для предотвращения спама через смену ника
  • Freeroam: добавлен антиспам и антифлуд
  • Freeroam: добавлена возможность делать закладки для координат местностей

Редактор

  • Добавлена информация о LOD'ах для того, чтобы упростить задачу их удаления
