StopSoundSlowly

From Multi Theft Auto: Wiki

This function stops the sound element slowly.

Syntax

int stopSoundSlowly( element sound )

Required Arguments

  • sound: The sound element that needs to be stopped.

Returns

Returns true if the sound was successfully stopped, false if the sound is not found.

Code

Click to collapse [-]
Clientside Script
function stopSoundSlowly (sElement)
    if not isElement(sElement) then return false end
    local sound_timer_quant = getSoundVolume(sElement)
    local slowlyStop = setTimer(
      function ()
        local soundVolume = getSoundVolume(sElement)
        setSoundVolume(sElement,soundVolume - 0.5)
	if soundVolume >= 0 then 
          stopSound(sElement) end
	end,400,sound_timer_quant*2
    )
end

Example

Click to collapse [-]
Clientside Script
addCommandHandler("stopS",
    function (command,sound)
        stopSoundSlowly (sound)
    end
)
