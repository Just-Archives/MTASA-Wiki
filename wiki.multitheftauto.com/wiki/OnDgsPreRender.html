OnDgsPreRender

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

This event is triggered every time before dgs renders.

Parameters

None

Source

The source of this event is the client's root element.

Example

This example shows that we can draw a rectangle under dgs gui. 

DGS = exports.dgs
edit = DGS:dgsCreateEdit(200,200,500,200,"test",false)

function drawAboveDGS ()
	dxDrawRectangle(150,150,600,300,tocolor(0,0,0,200),true) -- POSTGUI is always set to true
end
addEventHandler ( "onDgsPreRender", root, drawAboveDGS )
