Esempio di lista GUI.

La classe lista GUI rappresenta le liste di opzioni nella GUI di MTA:DM. Può essere usato ad esempio per creare delle liste di opzioni o per visualizzare delle informazioni sui giocatori (vedi la risorsa scoreboard).

Il tipo di elemento di questa classe è "gui-gridlist".
