Questa funzione restituisce l'icona che un segnalino attualmente ha

Sintassi

int getBlipIcon ( blip theBlip )

Argomenti richiesti

  • theBlip: il segnalino di cui stiamo per ricevere il numero di icona.

Esempio

Questo esempio trova tutti i segnalini che esistono e li imposta tutti all'icona di default. 

-- Recupera una tabella contenente tutti i segnalini che esistono
blips = getElementsByType ( "blip" )
-- Scorre l'elenco, e memorizza i segnalini dalla tabella nella variabile blipValue
for blipKey, blipValue in ipairs(blips) do
	-- Recupera l'icona del segnalino nella variabile 'blipIcon'
	blipIcon = getBlipIcon ( blipValue )
	-- Se l'icona del segnalino non è già quella di default
	if ( blipIcon ~= 0 ) then
		-- Imposta l'icona del segnalino di default
		setBlipIcon ( blipValue, 0 )
	end
end
