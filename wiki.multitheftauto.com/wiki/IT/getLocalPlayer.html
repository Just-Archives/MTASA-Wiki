Questa funzione restituisce l'elemento player del client che sta eseguendo lo script.

Sintassi

player getLocalPlayer ( )

Valori restituiti

Restituisce l'elemento player del client attuale.

Esempio

Click to collapse [-]
Client

Questo esempio scrive l'attuale posizione del giocatore nella console. 

local localPlayer = getLocalPlayer ( )

function mostraPosizioneAttuale ( )
	-- ottiene l'attuale posizione del giocatore
	local px, py, pz = getElementPosition ( localPlayer )
	-- la scrive nella console
	outputConsole ( "La tua attuale posizione: " .. px .. " " .. py .. " " .. pz )
end
