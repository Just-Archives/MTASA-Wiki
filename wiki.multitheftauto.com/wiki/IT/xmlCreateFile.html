IT/xmlCreateFile

From Multi Theft Auto: Wiki
Jump to navigation Jump to search

Questa funzione crea un nuovo documento XML, che potrà poi essere salvato su file tramite xmlSaveFile. Questa funzione sovrascriverà il file precedente se esso esiste già.

Sintassi

Click to collapse [-]
Server
xmlnode xmlCreateFile ( string filename, string rootNodeName, [ resource inResource = getThisResource() ] )

Argomenti richiesti

  • filename: il nome del documento XML che vuoi creare.
  • rootNodeName: il nome del nodo root del docoumento XML.

Argomenti opzionali

NOTA: Quando usi argomenti opzionali, devi fornire tutti gli altri argomenti prima di quello che vuoi utilizzare. Per altre informazioni consulta la pagina Argomenti opzionali.
  • inResource: la risorsa la cui directory verrà usata come root per il nuovo file. Notare che se viene specificata una risorsa esterna, quella che chiama la funzione deve avere accesso alla proprietà general.ModifyOtherObjects nell'ACL.
Click to collapse [-]
Client
xmlnode xmlCreateFile ( string filename, string rootNodeName )

Argomenti richiesti

  • filename: il nome del documento XML che vuoi creare.
  • rootNodeName: il nome del nodo root del docoumento XML.

Valori restituiti

Ritorna il nodo root del file XML se è stato creato con successo, false altrimenti.

Esempio

Click to expand [+]
Client & server

Questo esempio crea un documento XML di nome docuProva.xml, la cui root è chiamata rootprova e lo salva nella variabile docuXML. 

docuXML = xmlCreateFile("docuProva.xml", "rootprova")
