IT/fadeCamera

From Multi Theft Auto: Wiki
Jump to navigation Jump to search
« Torna alla Pagina principale italiana ▇▇▇.


Questa funzione dissolverà la telecamera di un giocatore in un colore o la riporterà alla normalità dopo un determinato periodo di tempo. Questo influenzerà anche il volume del suono del giocatore (dissolto al 50% = 50% del volume, dissolvenza totale = nessun suono).

Sintassi

Click to collapse [-]
Server & client
bool fadeCamera ( player thePlayer, bool fadeIn, [ float timeToFade = 1.0, int red = 0, int green = 0, int blue = 0 ] )

Argomenti Richiesti

  • thePlayer: Il giocatore la cui telecamera deve dissolversi.
  • fadeIn: Se la telecamera deve dissolversi in apertura (true) o in chiusura (false).

Argomenti Opzionali

NOTA: Quando usi argomenti opzionali, devi fornire tutti gli altri argomenti prima di quello che vuoi utilizzare. Per altre informazioni consulta la pagina Argomenti opzionali.
  • timeToFade: Il numero di secondi che impiegherà per dissolversi.
  • red: La quantità di rosso del colore nel quale la telecamera si dissolverà (0 - 255). Non necessario per dissolvenza in apertura.
  • green: La quantità di verde del colore nel quale la telecamera si dissolverà (0 - 255). Non necessario per dissolvenza in apertura.
  • blue: La quantità di blu del colore nel quale la telecamera si dissolverà (0 - 255). Non necessario per dissolvenza in apertura.

Valori restituiti

Ritorna true se la telecamera è stata dissolta con successo, false se sono stati passati argomenti non validi alla funzione.

Esempio

Click to expand [+]
Server & client

Quando un giocatore subisce danni, attiva un rapido effetto di dissolvenza in rosso sul suo schermo. 

function addRednessOnDamage ( )
      fadeCamera ( source, false, 1.0, 255, 0, 0 )         -- dissolve la telecamera del giocatore sul rosso per un periodo di un secondo
      setTimer ( fadeCamera, 500, 1, source, true, 0.5 )   -- non permette di dissolversi completamente, interrompe la dissolvenza e la riporta alla normalità
end
addEventHandler ( "onPlayerDamage", getRootElement(), addRednessOnDamage )
